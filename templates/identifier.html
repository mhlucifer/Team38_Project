About the Cane Toad Identifier

The Cane Toad Identifier is a tool designed to help identify cane toads (Rhinella marina) from images.

How to Use

  1. Click on the "Upload image here" button or box to select an image file from your device.
  2. Once an image is selected, it will be automatically uploaded and processed.
  3. The uploaded image will be displayed as a preview below the upload box.
  4. Click on the "Submit" button to initiate the identification process.
  5. The identification results will be shown in the "Result" section below.
  6. Identified a cane toad near you? Head to our handling guide for next steps.

Note: Ensure that the uploaded image contains clear and visible details to improve identification accuracy.

Disclaimer: This result is calculated based on a predictive machine learning model and is not a substitute for expert's advice.
